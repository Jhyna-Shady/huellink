Reportar mascota perdida

Registra los datos de tu mascota extraviada para que ciudadanos, rescatistas y refugios puedan ayudarte a encontrarla.

Datos importantes

Completa la información con el mayor detalle posible. La ubicación debe ser aproximada para proteger tus datos personales.

  • ✔ Agrega una foto clara de la mascota.
  • ✔ Indica la zona donde se perdió.
  • ✔ Describe color, tamaño y señales particulares.
  • ✔ Usa referencias fáciles de ubicar.

🔒 Seguridad

Huellink recomienda no publicar direcciones exactas de vivienda. Usa una zona, avenida o punto de referencia.
